# Changelog

Add one new line per completed task.

Format:

`[TYPE] short direct summary`

Allowed prefixes:

- `[FEATURE]`
- `[FIX]`
- `[REFACTOR]`
- `[DOCS]`
- `[TEST]`
- `[BUILD]`
- `[CHORE]`

Rules:

- one line only per entry
- no paragraphs
- no bullets inside entries
- keep the summary direct and specific
- append new entries to the end of the file
